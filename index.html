Galaxy Rush is a "play-to-earn" platform where users can join various online game tournaments, compete against others, and win real money. The app is built around a wallet system, allowing users to add funds, pay entry fees for matches, and withdraw their winnings.

Key Features

User Authentication & Security:

  • Secure login and sign-up system with username and password.
  • A 4-digit "Thunder PIN" is required for confirming sensitive transactions like sending money.
  • Password change functionality is available within the user's profile.

Server-Based Architecture:

  • Users must first connect to an active game server before logging in.
  • Servers have their own passwords and activation codes, suggesting different communities or game environments.
  • The app indicates server status (Online/Offline).

Wallet & Financial Transactions:

  • In-App Wallet: Manages a user's total balance, which is split into "Join Money" (deposited funds) and "Win Money" (earnings).
  • Add Money: Users can add funds to their wallet, seemingly via UPI by scanning a QR code.
  • Withdraw Money: Users can withdraw their "Win Money" to various payment methods like UPI, Paytm, PhonePe, or even as a Google Play Redeem Code.
  • Transaction History: A detailed log of all financial activities, including deposits, withdrawals, match entries, winnings, and gifts.

Gaming & Tournaments:

  • Multiple Game Modes: The platform supports various games, including "File Work" (task-based earning), "FREE Fire" (battle royale), "Ludo," and a "Mystery Box" for rewards.
  • Live Matches: The main dashboard displays a list of live and upcoming matches with details like prize pool, entry fee, and time.
  • Match Participation: Users can join matches by paying the entry fee from their wallet and submitting their in-game ID (like a Free Fire UID).
  • My Matches: A dedicated section shows all the matches a user has joined.
